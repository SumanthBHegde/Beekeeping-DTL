## Project Overview

This is a Design Thinking Lab (DTL) project for the MCA 2nd semester focused on designing an efficient beekeeping solution and hive box system.

**Project Name:** Efficient Beekeeping and Hive Box  
**Team:** Pavan Mohan Naik, Sumanth Bhaskar Hegde, Pramath Prakash Hegde  
**Guide:** Dr. Mohan Aradhya

## Project Structure

The repository is organized following the Design Thinking methodology with the following phases:

### 01-empathy/

Research and user discovery artifacts:

- **01-research/**: Field observation records, context/secondary research notes, guide PDFs, and media assets.
- **02-survey/**: Bee questionnaire design, survey notes, and responses spreadsheet.
- **03-interviews/**: Transcripts, audio files (`.mp3`/`.aac`), summaries, and media attachments for individual interviewees (Akshay Kedla, Dhruv Bhat, Harsha Hegde, Manjunath Hegde, Pavan, Sushanth Hegde, Ullas Naik).
- **04-empathy-map/**, **05-personas/**, **06-pain-points/**, **07-quotes/**, **08-insights/**: Phase outcomes and artifacts synthesized from research.

### 02-define/

Synthesis artifacts and problem definition:

- **01-affinity-mapping/**: Grouping of research observations.
- **02-pain-point-analysis.md**: Detailed breakdown of user challenges.
- **03-need-statements.md**: User-focused requirements.
- **04-problem-statement.md**: The core problem to be solved.
- **05-how-might-we.md**: Framed opportunity areas.
- **06-scope-decision.md**: Focus areas for the next phases.

### 03-ideate/

Ideation phase deliverables:

- **01-brainstorming/**: Idea log and general ideation notes.
- **02-dot-voting/**: Voting results to prioritize ideas.
- **03-concept-notes/**: Top concept details.
- **04-selection-matrix/**: Evaluation criteria and final concept selection.
- **05-storyboarding/**: Visual storytelling of the user journey before and after the proposed design.
- **06-innovation-and-impact/**: Detailed analysis of the project's innovation and its projected social, biological, and economic impact.

### 04-prototype/

Prototype development artifacts:

- **sketches/**: Hand-drawn concepts and design ideas.
- **wireframes/**: Digital layout/interaction wireframes.
- **prototype-versions/**: Physical and digital prototype iterations.

### 05-test/

Testing and validation phase:

- **test-plan/**: Protocols and setup for user testing.
- **feedback/**: Collected feedback from user tests.
- **iterations/**: Design changes based on feedback.

## Getting Started

To explore this project:

1. Navigate through the phase folders in order (01 в†’ 05)
2. Start with the `00-overview.md` files in each phase
3. Review the research artifacts and synthesis documents
