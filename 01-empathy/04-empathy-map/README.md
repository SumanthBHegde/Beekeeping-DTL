# Empathy Map

Based on our user interviews across different beekeeper personas (beginners, commercial farmers, and FPO managers) as well as the end-consumer, here is a synthesized empathy map highlighting their experiences.

## Producers (Farmers & Beekeepers)

| **Says**                                                                                   | **Thinks**                                                                           |
| :----------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------- |
| - "If we don't feed them or shift them during the off-season, they will just fly away."    | - "Is my colony sick? I can't look inside without disturbing them."                  |
| - "Heat is the biggest problem. If the sun hits the box directly, they won't store honey." | - "High-tech solutions like Flow Hive are too expensive and not practical."          |
| - "A transparent window would be perfect... keeping me safe and the bees undisturbed."     | - "I wish there was a way to maintain the bee space so the frames don't get glued."  |
| - "Flow hives at ₹25k are impossible for our farmers. Cost-effectiveness is everything."   | - "By-products like Royal Jelly could be profitable if we had the right technology." |
| - "My biggest fear is getting stung. If I open the box, they will attack."                 | - "Opening the box ruins the hive's temperature for two days."                       |

| **Does**                                                                                 | **Feels**                                                                                                 |
| :--------------------------------------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------- |
| - Relocates bee boxes seasonally to find adequate flora.                                 | - **Anxious/Fearful:** Beginners are terrified of being stung during routine maintenance.                 |
| - Feeds sugar syrup manually during the monsoons and off-seasons.                        | - **Intimidated:** Novices feel overwhelmed by the lack of training and high mortality rates of colonies. |
| - Hangs dead wasps near the hive to deter predatory wasps.                               | - **Frustrated:** Dealing with heavily glued frames (burr comb) and disrupted bee spaces.                 |
| - Uses a shared centrifuge to batch-extract honey instead of expensive individual tools. | - **Demotivated:** When colonies abscond due to bad weather or unrecognized diseases.                     |
| - Tries to avoid opening the box during extreme weather (monsoons/summer).               | - **Hopeful:** Excited about potential low-cost innovations (like external feeders or modular boxes).     |

## Conscious Consumers (End-Buyers)

| **Says**                                                                   | **Thinks**                                                                        |
| :------------------------------------------------------------------------- | :-------------------------------------------------------------------------------- |
| - "I want pure, unadulterated honey directly from the source."             | - "Is this store-bought honey mixed with sugar syrup?"                            |
| - "I don't mind paying a premium if I know it helps the farmers directly." | - "Are there micro-plastics from modern artificial combs leaching into my honey?" |
| - "Where can I find authentic, organic farm-fresh honey?"                  | - "I wonder what other health products bees make besides just honey."             |

| **Does**                                                            | **Feels**                                                                                                 |
| :------------------------------------------------------------------ | :-------------------------------------------------------------------------------------------------------- |
| - Pays premium retail prices (₹800/kg+) for verified quality honey. | - **Skeptical:** Of mass-market, commercial honey brands.                                                 |
| - Relies heavily on word-of-mouth networks to find trusted farmers. | - **Disconnected:** Wants to support local agriculture but lacks a direct bridge to the producers.        |
| - Seeks out organic or farm-to-table markets.                       | - **Curious:** Eager to try natural health supplements like Royal Jelly or Pollen if educated about them. |
