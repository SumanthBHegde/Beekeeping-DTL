# Wireframes

This directory contains structural layouts, user interface (UI) designs, and interaction flows for the **Safe Inspection & Feeding Upgrade** tools.

## Contents
* **UI Layouts:** Designs for the web-based interactive visualiser's control panel (the bottom-left dark glass UI), button groups, and dimensions block.
* **Camera Flows:** Diagrams or notes explaining the user journey through the various camera presets (Default, Front, Back, Left, Top, Window, Orbit).
* **App Wireframes:** Any subsequent mobile app or progressive web app (PWA) wireframes related to monitoring the hive.

## Usage
* Store Figma exports, Balsamiq files, PDF wireframes, or structural diagrams here.
* These serve as the blueprint for the UI implementation seen in the `beehive-prototype.html` file.
