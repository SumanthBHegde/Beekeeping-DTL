# Component Sketches — Safe Inspection & Feeding Upgrade

This folder holds hand-drawn and annotated diagrams for every component of the prototype. Until physical scans are uploaded, this file provides fully annotated text-based descriptions that serve as sketching guides.

## Sketch S-01 — Front Face Layout

```
┌─────────────────────────────────────────────────────┐  ← 505 mm wide
│                                                     │
│         ┌──────────────────────┐                   │  ← Window cutout
│         │  180 mm × 100 mm     │  80 mm from       │    (centred)
│         │  [ACRYLIC PANEL]     │  bottom edge      │
│         │  UV-tinted 5 mm      │                   │
│         └──────────────────────┘                   │
│  ↑ 80 mm                                           │
├─────────────────────────────────────────────────────┤  ← Bottom edge (505 mm)
│         [ENTRANCE SLOT 10 mm × 300 mm]              │
└─────────────────────────────────────────────────────┘
                         ↓
              [FEEDER TRAY — EXTERNAL]
              Boardman-style, 30 mm × 20 mm trough
              500 mL–1 L inverted reservoir above
```

**Annotation notes:**
- The window is deliberately positioned above the entrance to avoid cold draughts from the feeder area entering through the window recess.
- The 80 mm gap below the window preserves the hive's structural bottom rail.

## Sketch S-02 — Window Cross-Section (Side View)

```
OUTER FACE                        INNER FACE
───────────────────────────────────────────────────
│ Hive wall │  Routed   │ Acrylic │   Air gap   │ COMB
│  18 mm    │  recess   │  5 mm   │  ≥6.35 mm   │
│  timber   │  12 mm    │ panel   │  (bee space)│
───────────────────────────────────────────────────
                ↑ Acrylic sits flush in routed channel
                  Sealed with food-safe silicone bead
                  on outer perimeter only
```

**Key dimensions:**
- Wall recess depth: 12 mm (into 18 mm wall → 6 mm wall remains behind acrylic)
- Acrylic setback from inner face: matches existing inner wall plane
- Bee space gap: ≥ 6.35 mm from acrylic inner surface to nearest comb/frame
- Silicone sealant: **food-safe, zero VOC** (e.g., Dowsil 732 or equivalent) — outer bead only, never inner

## Sketch S-03 — Sliding Shutter Detail

```
FRONT VIEW — SHUTTER CLOSED          FRONT VIEW — SHUTTER OPEN
┌────────────┐                        ┌────────────┐
│ [SHUTTER]  │ ← 3 mm PVC/wood        │            │  Shutter slides
│ 200×120 mm │   opaque panel         │ [ACRYLIC   │  upward into
│            │   slides in MDF        │  VISIBLE]  │  top channel
└────────────┘   grooves (4 mm)       └────────────┘
   ↓ Closed = zero light leak           ↑ Open = inspection view
```

**Shutter travel:** 110 mm upward stroke — full panel clears window area
**Groove material:** 4 mm MDF channel, waxed with beeswax (no synthetic lubricant)
**Stop pin:** 3 mm brass rod at top of channel prevents shutter over-travel

## Sketch S-05 — Insulation Wrap (Budget Tier)

```
TOP VIEW — HIVE BODY

     ┌──────────────────────────────────┐
     │ XPS foam 20 mm (sides + back)   │
     │  ┌────────────────────────┐     │
     │  │   HIVE BODY interior   │     │
     │  │   33–36°C target       │     │
     │  └────────────────────────┘     │
     │  FRONT: No insulation wrap      │
     │  (window + entrance must stay   │
     │   unobstructed)                 │
     └──────────────────────────────────┘
```

**XPS thickness:** 20 mm — R-value ≈ 0.55 m²·K/W (sufficient for Bengaluru winter nights)
**Ventilation:** Mesh ventilation strip retained at top of rear wall — breathable, not sealed
**Moisture:** XPS is closed-cell; no moisture absorption; mould risk eliminated
