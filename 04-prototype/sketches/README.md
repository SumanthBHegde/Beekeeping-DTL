# Sketches

This directory is designated for conceptual artwork, ideation sketches, and early visual explorations for the **Safe Inspection & Feeding Upgrade**.

## Purpose
* **Hand Sketches:** Scans or photos of whiteboard drawings, notebook sketches, or physical modeling notes detailing the dimensions and mechanisms (e.g., the sliding shutter tracks, the external funnel brackets).
* **Screenshots:** Reference screenshots of the 3D visualiser during early stages of development or showing specific camera angles/states for documentation purposes.
* **Component Concepts:** Early drawings of the insulation placement, the window placement relative to the bee space, and the flow frame drainage routes.

## Usage
* Place image files (`.png`, `.jpg`, etc.) here.
* If a sketch heavily influences a design decision in the prototype, reference it in the main project documentation.
