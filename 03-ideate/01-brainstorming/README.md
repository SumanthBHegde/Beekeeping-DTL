# 01 - Brainstorming

**Purpose:**
This folder captures the raw, unfiltered generation of ideas. It is the very first step in the Ideation phase.

**Contents:**
- `idea-log.md`: The master log of all generated ideas. 

**Methodology:**
To prevent "design fixation" (where a team latches onto the first acceptable idea), we utilized four distinct methodologies to force lateral thinking:
1. **Brainwriting 6-3-5:** For volume and iteration.
2. **SCAMPER:** For physically modifying existing constraints.
3. **Reverse Brainstorming:** For discovering robust defensive solutions against pests.
4. **Group Discussion:** For capturing standalone inspirations.

All ideas in this folder are directly mapped to the "How Might We" (HMW) questions defined in our previous research phase.
