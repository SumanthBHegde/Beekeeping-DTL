# 03 - Concept Notes

**Purpose:**
This folder transforms raw ideas into concrete, workable concepts.

**Contents:**
- `top-concepts.md`: A highly detailed breakdown of the winning ideas from the dot-voting phase.

**Methodology:**
An "idea" is just a sentence. A "concept" is an actionable blueprint. In this stage, we take the surviving ideas and expand them into structured notes that define exactly *how* the idea works, what specific user needs it addresses, its key features, and its inherent risks. These notes ensure the team fully understands the mechanics of a solution before spending time rigorously evaluating it in the Selection Matrix or building it in the Prototype phase.
